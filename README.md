# NutriScope – Personalized Nutritional Assistant

NutriScope is a personalized nutrition tracking web application that helps users monitor calories, protein, carbs, fats, fiber, and water intake.

## Features

• Personalized nutrition goals based on body stats  
• Food logging with IFCT and USDA database  
• Daily progress tracking  
• Weekly analytics and charts  
• AI Coach for smart recommendations  
• Profile and goal management  

## Tech Stack

Frontend: Next.js, TypeScript, Tailwind CSS  
Backend: FastAPI, Python  
Database: Supabase (PostgreSQL)  
Charts: Recharts  

## Folder Structure

frontend/ → Next.js frontend  
backend/ → FastAPI backend  
database/ → SQL scripts
